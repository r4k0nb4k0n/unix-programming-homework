# 10. 프로세스 관리
## 프로세스
* 프로세스
	* 실행중인 프로그램
	* 자원 소유, 상태를 가짐
* 프로세스 ID
	* **특정 시점**에서 프로세스의 유일한 식별자
	* 순차적으로 할당됨
		* 프로세스 A, B, C가 만들어지면 PID는 A < B < C(**항상 참은 아님**)
	* 특별한 프로세스 ID
		* 0번: swapper(또는 idle) process
			* 실행될 프로세스가 없을 때 커널이 실행하는 프로세스
		* 1번: init process
			* 시스템의 초기화 담당
			* 모든 프로세스의 조상 프로세스
## 프로세스의 계층 구조
* 리눅스의 프로세스는 부모 자식의 관계를 가짐
* 프로세스 그룹
* `pid_t`
