# 09. 비트 연산
## 비트와 비트 연산
* 비트
  * 컴퓨터에서 데이터를 저장하는 최소 단위
  * `0` or `1`
* 비트 연산
  * and: `&`
  * or: `|`
  * not: `~`
  * xor: `^`
* 쉬프트 연산
  * 오른쪽 쉬프트: `>>`
    * `2`로 나눈 효과
  * 왼쪽 쉬프트: `<<`
    * `2`를 곱한 효과
